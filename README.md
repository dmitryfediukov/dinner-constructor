# Конструктор обеда

## Идея приложения
В ресторане вкусно готовят, но нет ресурсов на составление комбо наборов.
Данное приложение должно уметь составлять комбо наборы, чтобы сотрудникам ресторана не требовалось
на это больших усилий.

## Функционал приложения
Приложение имеет следующие функции:
1. Добавление блюда с указанием его типа;
2. Генерация и вывод необходимого количества комбо наборов по указанным типам блюд.

## Классы приложения
Разработанное приложение имеет следующие классы:
1. **Main** - в данном классе происходит все взаимодействие с пользователем;
2. **DinnerConstructor** - класс, в котором реализована логика добавления блюд, проверка наличия типа блюд, 
случайный выбор блюда для набора, генерация комбо наборов.
